# COOK FARM — Landing

Frontend landing site scaffold using Vite + React + TypeScript + TailwindCSS + Framer Motion + Lucide.

Run:

```bash
npm install
npm run dev
```

This is a pure frontend landing; no backend integration.

## Documentation

- [Project Understanding](./PROJECT_UNDERSTANDING.md) – details about architecture, technologies, and file structure.
- [Deployment Instructions](./DEPLOYMENT.md) – steps for publishing to GitHub Pages.

A helper script (`deploy.sh`) is provided to automate the build and push process. Make it executable and run it from the project root:

```bash
chmod +x deploy.sh
./deploy.sh
```
